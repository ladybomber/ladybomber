NullDC Bear

Para fazer o download do emulador, entre no link abaixo:
https://github.com/RossenX/NullDC-BEAR/releases/tag/1.96c

Clique no NullDC.BEAR.exe como mostra na imagem:
Após fazer o download NÃO EXECUTE-O AINDA!, você vai criar uma pasta onde ficará o emulador em qualquer lugar do seu PC que não seja na pasta Downloads padrão do Windows, feito isso recorte o arquivo baixado e cole na pasta que você criou.
Abra o programa e instale:
1º Será perguntado se você deseja instalar o NullDC dentro da pasta atual, escolha a opção YES
2º Abrirá outra caixa dizendo que será adicionado um monte de arquivos na pasta atual junto com o instalador, escolha a opção YES
Ao finalizar, abrirá o Setup básico do NullDC, digite seu nick e clique no botão verde para adicionar o NullDC no Firewall do Windows e após acabar clique em Save.
Pronto, mas agora antes de continuar é necessário atualizar o emulador para que o download automático funcione, para isso clique aqui

O NullDC tem várias configurações, porém, só vou dizer o necessário para jogar!

Agora você vai no menu Free DLC, clique na aba(do console desejado) SNES ou digite(nome do jogo desejado) Super Bomberman 4
Agora vamos configurar o controle, lembre-se cada console que for jogar terá que configurar o controle uma vez, se for jogar apenas Super Nintendo, só precisará configurar uma vez.

1º Clique em Controls
2º Clique na aba SNES
3º Se você for jogar com um controle de videogame escolha-o, em seguida clique em cada botão do controle e configure
4º Salve
Algumas configurações opcionais, para ajustar a janela do jogo faça o seguinte:

1º Clique em Options >> Emulators
2º Clique no ícone do Super Nintendo
3º Na parte Window você escolhe o tamanho que caber em seu monitor
4º Upscaler muda a aparência pode deixar com mais qualidade
5º Stretch é o formato da tela do jogo
Pronto, faça um teste, abra o jogo offline e vê se está tudo ok!
Para criar um jogo com a possibilidade de alguém entrar e jogar com você, crie o jogo da seguinte forma:

1º Clique em Play
2º Escolha o jogo
3º No Server coloque Localhost
4º MultiTap coloca Port 2
5º Clique em LETS GO!
Para entrar em um jogo em andamento, basta clicar duas vezes no nome do player que está como Hosting ou Public, ou clique apenas uma vez e clique no botão Join/Spectate
Hosting é quem criou o jogo pelo Localhost,
Public é quem criou o jogo através de um servidor
Client são os jogadores que entraram no Hosting de alguém para jogar, você não vai conseguir entrar no jogo clicando sobre um Client, somente no Hosting ou Public


ATUALIZAR EMULADOR (21/09/2025)

É necessário você atualizar depois de baixar o emulador

Por que atualizar?
O download automatico dos jogos direto pelo emulador não estão funcionando, para corrigir basta fazer download do arquivo do console que deseja abaixo:

O arquivo para quem quer baixar jogos de Super Nintendo é esse:
SNES.freedlc

O arquivo para quem quer baixar jogos de Nintendo 64 é esse:
N64.freedlc

O arquivo para quem quer baixar jogos de Nintendinho é esse:
NES.freedlc

O arquivo para quem quer baixar jogos de Game Boy Advance é esse:
GBA.freedlc

O arquivo para quem quer baixar jogos de Game Boy Color é esse:
GBC.freedlc

O arquivo para quem quer baixar jogos de PC Engine é esse:
PCE.freedlc

O arquivo para quem quer baixar jogos de Sega Master System é esse:
SMS.freedlc

O arquivo para quem quer baixar jogos de Sega Saturn é esse:
Saturn.freedlc

O arquivo para quem quer baixar jogos de Playstation são esses:
PSX.freedlc
PSX_Asia.freedlc
PSX_PAL.freedlc

Após baixar, pegue o arquivo baixado(normalmente na pasta Downloads) e coloque ele dentro da pasta DLC que fica dentro da pasta do emulador NullDC BEAR, como mostra na imagem abaixo:



Após substituir os arquivos, basta ir no NullDC BEAR na aba Free DLC e baixe normalmente, continue nesta parte do tutorial clicando aqui
